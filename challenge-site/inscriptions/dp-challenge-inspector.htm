Satplication Architecture

Premise/Articles sat – contains the preface (reinscribed) + article manifest (JSON listing all articles/DPs)
Layout sat – CSS theme + structural layout inscribed as HTML
Script sats – modular JavaScript: Oracle (main loader), TimeTravel, Articles module, Data module, UI, Pagination, Tags, Sources
Config sat – sources JSON: key-value pairs defining the satplication's runtime behavior
Hook sat – BRC333 hook logic for dynamic behavior triggers
Styles sat – additional CSS/theme overrides
